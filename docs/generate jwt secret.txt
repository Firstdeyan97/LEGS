Langkah Men-generate JWT Secret yang Aman
Buka terminal Anda (Anda bisa membiarkan nodemon tetap berjalan dan membuka tab terminal baru, atau mematikannya sebentar dengan Ctrl + C).

Copas perintah satu baris ini lalu tekan Enter:

Bash
node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
Terminal akan langsung mencetak string acak (hexadecimal) yang sangat kuat sepanjang 128 karakter. Contoh keluarannya akan terlihat seperti ini:
8f3a9b2c7d8e9f0a1b2c3d4e5f6a7b8c9d0e1f2a3b4c5d6e7f8a9b0c1d2e3f4a...

Blok dan Copy seluruh string hasil tersebut.

Buka kembali file backend/.env Anda, lalu paste hasilnya untuk menggantikan nilai di variabel JWT_SECRET.
